## 🤝 介绍

本插件作用与Astrbot发送消息前，提供了一系列增强功能：

1. 🖼️ 图片外显：把“[图片]”替换成随机金句。  
2. 🛡️ 报错拦截：命中关键词立即屏蔽或私聊转发给管理员，群里不再刷屏。  
3. 🧠 仅处理 LLM：后续所有“智能”步骤只针对大模型返回的消息生效，建议开启。  
4. 🤖 人机拦截：复读、AI 腔一键拦，关键词可自定义。  
5. @️ 假@解析：把“@昵称”自动转成真@，再按概率决定留不留。  
6. 🧹 文本清洗：括号、emoji、前缀后缀一键剥，短消息才生效。  
7. 🎙️ TTS：超长文本自动转语音，20 种角色可选，中转群隐藏 AI 标识。  
8. 💬 智能引用：别人插嘴超过 N 条才自动带上原消息，避免刷屏。  
9. 🖨️ 长文转图：超阈值自动渲染成图片，支持分页、样式自定义。  
10. 📦 合并转发：继续超长就自动合并转发，节点昵称可自定义。  
11. ↩️ 自动撤回：检测到关键词 N 秒后撤回，防隐私/测试泄露。
